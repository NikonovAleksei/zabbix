# Active Directory monitoring with Zabbix  

## [AD DS Health and Performance v 2.12](https://github.com/NikonovAleksei/zabbix/wiki/AD-DS-Health-and-Performance)
This template provides comprehensive health, performance, and availability monitoring for Microsoft Active Directory Domain Services (AD DS) and related DNS components.

It is designed to detect service degradation, replication issues, authentication latency, resource exhaustion, and configuration anomalies that directly impact domain stability and authentication reliability.

The template combines performance counters, service checks, and state-based logic to deliver both real-time alerts and trend-based early warning signals, making it suitable for production enterprise environments.

The folder contains zabbix template for Active Directory and based on MS documents:
- Troubleshooting Active Directory Domain Services performance
- Capacity Planning for Active Directory Domain Services
- How to do performance tuning for NTLM authentication by using the MaxConcurrentApi setting

## [AD DS Monitoring and Attack Detection v 2.00](https://github.com/NikonovAleksei/zabbix/wiki/AD-DS-Monitoring-and-Attack-Detection)
This template provides comprehensive monitoring and attack detection for Microsoft Active Directory Domain Services (AD DS).

It focuses on security-relevant events, directory service health, and domain controller availability, allowing early detection of misconfigurations, replication issues, and common attack techniques targeting AD environments.

Template based on MS documents: 
- Best Practices for Securing Active Directory
